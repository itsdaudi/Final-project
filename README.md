# Rahajipe Resort

A modern hotel/resort booking website built with HTML, CSS, and JavaScript. Experience luxury and tranquility at Rahajipe Resort.

## Features

- **Home Page**: Hero section with resort overview, image gallery showcasing amenities, and recent booking preview
- **Booking System**: Complete booking form with form validation
  - Full Name
  - Email Address
  - Preferred Date
  - Resort Package selection
  - Number of Participants
  - Special Requests
- **My Bookings**: View all your bookings, delete individual bookings, or clear all bookings
- **Form Validation**: Real-time validation with error messages
- **Data Persistence**: All bookings are stored in localStorage

## How to Use

1. Open `index.html` in a web browser
2. Navigate to "BOOK NOW" to make a reservation
3. Fill in the booking form with your details
4. Submit to save your booking
5. View your bookings on the "My Bookings" page
6. You can delete bookings as needed
